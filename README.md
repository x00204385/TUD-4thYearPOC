# TUD-4thYearPOC
Some proof of concept work in support of phase 1 of the TUD 4th year project course. These are based on tutorials for the technologies
being used in the project. 

The proofs of concepts are:
* [vpc-tf](/vpc-tf) Build infrastructure using Terraform and deploy a web server in a load balanced configuration
* [eks-cmd](/eks-cmd) Build a managed Kubernetes cluster using the eksctl command line and deploy a web server to it
* [eks-tf](/eks-tf) Build a managed Kubernetes cluster using the Terraform 
* [medium-manifests](/medium-manifests) Build public cloud infrastructure using CrossPlane
